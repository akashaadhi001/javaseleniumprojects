# Java Selenium Maven Project

This project is a simple Maven setup for Java Selenium automation.

## Requirements
- Java 17+
- Maven 3.9+
- A compatible browser driver (for example ChromeDriver) if you run browser-based tests

## Run tests
```bash
mvn test
```

## Sample structure
- `src/test/java` contains example tests
- `pom.xml` defines Selenium and JUnit dependencies
